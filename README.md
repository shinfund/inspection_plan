# 월간 점검계획서 생성기

청하터널 전기안전관리 월간 점검계획서를 자동으로 작성하는 웹 도구입니다.

## 사용 방법

1. `index.html`을 브라우저에서 열거나 GitHub Pages URL에 접속합니다.
2. 연도와 월을 선택합니다.
3. 템플릿 엑셀 파일을 업로드합니다.
4. **생성** 버튼을 클릭하면 완성된 점검계획서가 다운로드됩니다.

## GitHub Pages 배포

1. GitHub에서 새 저장소 생성 (예: `username.github.io` 또는 일반 저장소)
2. 이 폴더의 파일들을 저장소에 업로드
3. Settings → Pages → Source: `main` 브랜치로 설정
4. `https://username.github.io` 또는 `https://username.github.io/저장소명` 으로 접속

## 파일 구조

```
github-pages/
├── index.html    # 점검계획서 생성기 (메인 페이지)
└── README.md     # 프로젝트 설명
```
